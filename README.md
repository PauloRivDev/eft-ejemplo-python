# Comandos básicos de git en shell

## Configuración

### Comando para identificar correo
git config (--global) user.email "you@example.com"

### Comando para identificar usuario
git config --global user.name "Your Name"


## Comandos

### Agregar un solo archivo al commit
git add <archivo_nombre>

### Agregar todos los cambios que realice
git add .

### Como en github se va a llamar mis cambios
git commit -m "nombre-de-los-cambios"

### Comando para subir todo a github
git push
